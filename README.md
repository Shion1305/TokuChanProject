# 匿名で質問ができるDiscord Bot「匿ちゃん」

## 概要

理工学部情報工EPのDiscordサーバー用に作成した匿名質問BOTです。

## ざっくりとした仕様

Bot用のアカウントキーを登録して、Botアカウントにメッセージを送信すると、設定したチャンネルにメッセージを転送します。

匿名で送信できるほか、送信取り消しや送信IDの変更もできるようになっています!
